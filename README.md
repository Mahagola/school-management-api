💻 Local Setup
Clone the repository.

Run npm install to install dependencies.

Create a .env file with your database credentials (DB_HOST, DB_USER, DB_PASSWORD, DB_NAME, PORT).

Run node index.js to start the server on port 3000.


***

Once you drop that into your repository, your project is 100% complete and ready to hand in.
